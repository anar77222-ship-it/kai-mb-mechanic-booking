Kai MB Mechanic Booking App (Local)

1) Open PowerShell in your folder (example: E:\kai_booking_app)

2) Install:
   python -m pip install --upgrade pip
   python -m pip install -r requirements.txt

3) Run:
   python -m streamlit run app.py

Admin password:
- Set env var KAI_ADMIN_PASSWORD (recommended), or edit ADMIN_PASSWORD in app.py
